# Videos Folder

Place your chatbot demonstration videos here.

**Options:**
1. **Upload video files directly** (recommended for short demos)
   - MP4 format preferred
   - Keep under 50MB if possible
   - Name descriptively: `chatbot-demo.mp4`, `conversation-flow.mp4`

2. **Use external video hosting** (recommended for longer videos)
   - Upload to YouTube, Loom, or similar
   - Create a text file with the link: `video-link.txt`
   - Update the project README with the embedded/linked video
