# Screenshots Folder

Place your chatbot screenshots here.

Recommended screenshots to include:
- Complete chatbot flow diagram
- Example conversations showing different scenarios
- Node configurations for key logic
- Integration setups
- Response templates

**Naming Convention:**
- Use descriptive names: `chatbot-flow.png`, `conversation-example.png`, `ai-config.png`
- Use PNG or JPG format
- Keep file sizes reasonable (compress if needed)
