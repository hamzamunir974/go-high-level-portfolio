# Screenshots Folder

Place your workflow screenshots here.

Recommended screenshots to include:
- Full workflow overview showing all nodes
- Close-ups of important node configurations
- Sample input/output data
- Error handling flows
- Results or outcomes

**Naming Convention:**
- Use descriptive names: `workflow-overview.png`, `node-config.png`, `results.png`
- Use PNG or JPG format
- Keep file sizes reasonable (compress if needed)
